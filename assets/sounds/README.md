# Taxi Driver App - Audio Assets

Bu papkada ilovaning ovoz fayllar joylashgan.

## Ovoz fayllari:

1. **new_order.ogg** - Yangi buyurtma kelganda chalinadigan ovoz (48.7 KB)
2. **order_accepted.mp3** - Buyurtma qabul qilinganda chalinadigan ovoz (62 KB)
3. **arriving.mp3** - Manzilga yaqinlashganda chalinadigan ovoz (558 KB)
4. **success.mp3** - Safar muvaffaqiyatli yakunlanganda chalinadigan ovoz (56 KB)

## Ovoz fayl formatlari:
- Format: MP3
- Hajm: 50KB gacha (kichik hajmda)
- Davomiyligi: 1-3 soniya
- Sifat: 128kbps

## Ovoz fayllari manbasi:
Ovoz fayllarini quyidagi bepul saytlardan topishingiz mumkin:
- https://pixabay.com/sound-effects/
- https://www.zapsplat.com/
- https://freesound.org/

## O'rnatish:
1. Ovoz fayllarini yuklab oling
2. Fayllarni `assets/sounds/` papkaga joylashtiring
3. Fayl nomlari yuqoridagi ro'yxatdagi nomlarga mos kelishi kerak

## Sozlamalar:
Foydalanuvchilar sozlamalar sahifasidan ovozlarni yoqish/o'chirish mumkin:
- Sozlamalar → Ovoz → Bildirishnoma ovozlari

## Dasturiy kod:
Ovozlar `SoundService` klassi orqali boshqariladi:
- `lib/core/utils/sound_service.dart`
